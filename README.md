Teste de classificação com iris dataset comparando resultado da predição entre dois modelos
Decision Tree Classifier
K-Nearest Neighbors (KNN)

O modelo já pronto do scikit-learn me permitiu verificar qual era o melhor especificamente para esse problema
Após quebrar a cabeça um pouco, comecei fazendo com Árvore de Decisão mas acabei testando com o KNN após algumas pesquisas não triviais em relação
a problemas de classificação.

O resultado: O modelo KNN se saiu um pouco melhor com acurácia de 93%
